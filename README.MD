# generator-webpack-webapp

## 开始使用

推荐使用内网, 安装cnpm， 如果已安装跳过这一步
```
   $ npm install -g cnpm --registry=http://registry.npm.taobao.org

   $ cnpm set registry http://172.18.22.198:7001
```

安装yo
```
   $ cnpm install -g yo
```

安装生成器
```
  $  cnpm install -g generator-webpack-webapp
```

使用
```
  $ yo webpack-webapp
```
